1. Установить: Python 3.9+, React + Vite, PostgreSQL
2. Установить библиотеки для python описанные в файле requirements.txt
3. Создать базу данных описанную в db.py, запустив нужный скрипт
4. выполнить npm install в папке frontend
5. Запустить main.py и в папке frontend выполнить npm run dev 
Дождаться скачивания всех моделей (Около 50 гб), это может занять много времени
